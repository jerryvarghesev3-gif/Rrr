Description (Simple)

This story focuses on extracting the bundled files (bundle.zip) during installation and setting up the launcher (launcher.exe) so the application can start correctly after installation.

⸻

✅ Acceptance Criteria (4–5 Clear Points)
	•	bundle.zip is extracted successfully to the target installation folder
	•	Extracted files are placed correctly with no missing or corrupted files
	•	launcher.exe is installed and configured properly
	•	Installer performs extraction → setup in correct sequence
	•	Application launches successfully using the installed launcher
