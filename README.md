Description (Simple)

This story focuses on building the installer GUI flow, starting from user permission (when launching the .exe) and guiding the user step-by-step through the installation process with clear progress updates and final status.

⸻

✅ Acceptance Criteria (4–5 Clear Points)
	•	Installer shows permission prompt when launched
	•	Installation starts only after user approval
	•	GUI guides user through installation steps clearly
	•	Progress bar shows real-time installation status
	•	Installation completes with clear success or failure message
