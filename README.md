Description (Simple)

This story focuses on adding required application files (bundle.zip and launcher.exe) into the installer and configuring the packaging so they are included and available during installation and execution.

✅ Acceptance Criteria (4–5 Clear Points)

	•	Packaged installer (.exe) contains both components correctly
	•	Installation/extraction makes these files available for use
	•	Installer build completes successfully without errors
