
Description

Create the base installer project structure and configure the initial build setup (Part-1).
This includes setting up the installer framework, defining project structure, and enabling the first successful build to validate the setup.

⸻

✅ Acceptance Criteria
	•	Installer project is created using the selected installer framework/tool
	•	Project structure is defined (folders for binaries, configs, bundle, scripts, etc.)
	•	Build configuration (debug/release) is set up correctly
	•	Basic installer metadata configured (name, version, output path)
	•	Initial installer build is successfully generated without errors
	•	Output installer executable (.exe) is created
	•	Build process is repeatable and stable

⸻

✅ Output
	•	Installer project repository initialized with proper structure
	•	Initial installer build (.exe) generated successfully
	•	Build configuration ready for further feature additions (bundle, launcher, workflows)
	•	Foundation setup completed for Part-2 enhancements
