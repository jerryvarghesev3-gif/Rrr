
Description

Implement installation GUI workflow that starts from user permission (on launching the .exe) and guides through installation steps including dependency setup and progress tracking.
This ensures a smooth, user-driven installation experience with proper permission handling and visibility into installation progress.

⸻

✅ Acceptance Criteria
	•	On launching the installer (.exe), user permission prompt is displayed and handled correctly
	•	Installation proceeds only after user grants required permissions
	•	GUI guides user through installation steps in a clear and sequential manner
	•	Required dependencies are installed in respective system paths successfully
	•	Installation progress is displayed through a progress bar with real-time updates
	•	Errors during installation are handled and communicated clearly to the user
	•	Installation completes successfully with proper status indication

⸻

✅ Output
	•	Installer GUI flow executed from permission prompt to completion
	•	Dependencies installed and configured in required locations
	•	Installation progress visibly tracked via progress bar
	•	User receives clear success/failure status at end of installation
