Description (Simple)

This story focuses on understanding and implementing the installation and extraction workflow based on the existing service tool. It ensures that required application files are packaged, extracted, and executed correctly as part of the installer flow.

⸻

✅ Acceptance Criteria (4–5 Clear Points)
	•	Required application files are included in the installer package
	•	Installer extracts files correctly to the target location
	•	Installation flow runs automatically without manual steps
	•	Application/launcher executes successfully after installation
	•	End-to-end installation completes without errors
