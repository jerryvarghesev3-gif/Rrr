Description

Install and register the launcher after successful bundle extraction.
This step ensures the extracted launcher component is placed in the correct location, configured for execution, and made ready for the next application startup phase.

Acceptance Criteria
	•	Launcher installation starts only after bundle extraction completes successfully
	•	launcher.exe is copied/placed in the correct target location
	•	Required launcher-related files/configuration are available after installation
	•	Launcher installation completes without errors
	•	Installed launcher is accessible and ready for execution
	•	Installation result is captured and reported for next workflow step

Output
	•	Launcher installed successfully after bundle extraction
	•	Launcher files placed in correct execution path
	•	Launcher environment prepared for startup
	•	Installation status available for workflow continuation
